What is HOCO? High-Resolution Convective Outlooks (HOCO) utilizes advanced ensemble forecasting by aggregating and weighting top-tier meteorological models. It dynamically renders 3D risk assessments for severe weather and convective activity across mapped regions.

How to Use HOCO

  1. Layout: Use the sidebar to split your view into 1, 2, or 4 maps for easy comparison.
  2. Archive: Select a past forecast run from the dropdown and assign it to a specific map to view the 3D data grid.
  3. Execution (Normal+): Define a region, time period, and hazards, then generate a new custom ensemble.
  4. Analysis: Rotate the map (Right-Click + Drag) to view risk elevation. Click specific grid zones to see precise risk index data.
